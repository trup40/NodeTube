## 🇺🇸 English

First off, thank you for considering contributing to NodeTube! 🚀

As an open-source project, we welcome community contributions. Whether it's fixing a bug, adding a new feature, or improving documentation, your help is appreciated.

### How Can I Contribute?

#### 1. Report Bugs
If you find a bug, please open an **Issue** and include:
* Your operating system and browser.
* Steps to reproduce the bug.
* Expected behavior vs. what actually happened.

#### 2. Suggest Features
Have a great idea for NodeTube? Open an **Issue** with the tag `enhancement` and describe your idea before writing any code. This helps us discuss the best way to implement it.

#### 3. Submit a Pull Request (PR)
If you want to contribute code, follow these steps:
1. **Fork** the repository.
2. Create a new branch for your feature (`git checkout -b feature/amazing-feature`).
3. Make your changes and test them locally.
4. Commit your changes (`git commit -m 'Add some amazing feature'`).
5. Push to the branch (`git push origin feature/amazing-feature`).
6. Open a **Pull Request** and describe the changes you made.

### Development Setup

To run NodeTube locally for development:

1. Clone your fork (or the main repo):
   ```bash
   git clone https://github.com/trup40/NodeTube.git
   ```
2. Navigate to the project directory and install dependencies:
   ```bash
   cd NodeTube
   npm install
   ```
3. **Important:** Make sure you have `yt-dlp.exe` downloaded and placed in the root directory of the project.
4. Start the server:
   ```bash
   npm start
   ```
5. Open your browser and visit `http://localhost:3000`.

Thanks for helping make NodeTube better! 🎵

## 🇹🇷 Türkçe

Öncelikle NodeTube'a katkıda bulunmayı düşündüğünüz için teşekkürler! 🚀

Açık kaynaklı bir proje olarak topluluk katkılarını memnuniyetle karşılıyoruz. İster bir hatayı düzeltmek, ister yeni bir özellik eklemek veya belgeleri iyileştirmek olsun, yardımınız bizim için değerlidir.

### Nasıl Katkıda Bulunabilirim?

#### 1. Hata Bildirin (Report Bugs)
Eğer bir hata bulursanız, lütfen bir **Issue (Sorun)** açın ve şunları ekleyin:
* İşletim sisteminiz ve tarayıcınız.
* Hatayı yeniden üretme adımları.
* Beklenen durum ve gerçekte olan şey.

#### 2. Özellik Önerin (Suggest Features)
NodeTube için harika bir fikriniz mi var? Kod yazmaya başlamadan önce fikrinizi açıklayan ve `enhancement` (geliştirme) etiketine sahip bir **Issue** açın. Bu, onu uygulamanın en iyi yolunu tartışmamıza yardımcı olur.

#### 3. Pull Request (PR) Gönderin
Eğer kod ile katkıda bulunmak istiyorsanız şu adımları izleyin:
1. Repoyu **Fork**'layın.
2. Özelliğiniz için yeni bir dal (branch) oluşturun (`git checkout -b ozellik/harika-ozellik`).
3. Değişikliklerinizi yapın ve yerel olarak test edin.
4. Değişikliklerinizi commit'leyin (`git commit -m 'Harika bir ozellik eklendi'`).
5. Dalınıza push'layın (`git push origin ozellik/harika-ozellik`).
6. Bir **Pull Request** açın ve yaptığınız değişiklikleri açıklayın.

### Geliştirme Ortamı Kurulumu

Geliştirme için NodeTube'u yerel ortamınızda çalıştırmak için:

1. Repoyu klonlayın:
   ```bash
   git clone https://github.com/trup40/NodeTube.git
   ```
2. Proje dizinine gidin ve bağımlılıkları yükleyin:
   ```bash
   cd NodeTube
   npm install
   ```
3. **Önemli:** `yt-dlp.exe` dosyasını indirip projenin ana dizinine yerleştirdiğinizden emin olun.
4. Sunucuyu başlatın:
   ```bash
   npm start
   ```
5. Tarayıcınızı açın ve `http://localhost:3000` adresine gidin.

NodeTube'u daha iyi bir hale getirmeye yardımcı olduğunuz için teşekkürler! 🎵
